# Fantastic Umbrella — Website

This repository contains a starter static website (index.html, styles.css, script.js).

How to use

- View the site locally: open index.html in a browser.
- Publish with GitHub Pages: go to the repository Settings → Pages, choose the `main` branch and `/ (root)`, then Save. The site will be available at https://<your-username>.github.io/fantastic-umbrella/ after it builds.
